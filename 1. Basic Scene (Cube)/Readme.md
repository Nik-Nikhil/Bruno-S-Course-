# 🧊 Basic Scene (Cubes)

A simple 3D scene featuring two cubes — one solid and one wireframe — built using [React Three Fiber (R3F)].
## 🧰 Tech Stack

- React
- React Three Fiber
- Three.js
- Vite

## 📦 Installation

 - npm install
 
## 🖥️ Run Locally

 - npm run dev
